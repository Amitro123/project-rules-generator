# Writing Skills

## Purpose
Meta-skill for creating new skills.

## Auto-Trigger
- When creating a new skill.
- When the user asks for guidance on creating skills.

## Process
1.  **Define Purpose**: Clearly state the purpose of the skill.
2.  **Define Auto-Trigger**: Specify conditions under which the skill should activate.
3.  **Define Process**: Outline the steps to execute the skill.
4.  **Define Anti-Patterns**: Identify common mistakes.
5.  **Review**: Ensure clarity and completeness.

## Anti-Patterns
- Creating a skill without a clear purpose.
- Missing auto-trigger conditions.
- Vague or incomplete process steps.
