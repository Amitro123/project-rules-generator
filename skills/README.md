# Skills System

This directory contains the skills available to the project.

## Structure
- `builtin/`: Skills provided by default.
- `awesome/`: Skills from famous projects.
- `learned/`: Skills generated by the project.

## Skills Format
Each skill is defined in a `SKILL.md` file within its directory.
The file should contain:
- Purpose
- Auto-Trigger
- Process
- Anti-Patterns
