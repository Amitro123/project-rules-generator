# Sample Project

A demo project for testing the rules generator. This shows basic functionality.

## Overview

This is a Python-based CLI tool that processes data files and outputs reports. It supports multiple output formats including JSON, CSV, and Markdown.

## Features

✅ **Fast Processing** - Handles 10,000+ files per minute
✅ **Multiple Formats** - JSON, CSV, Markdown output
✅ **Flexible Config** - YAML-based configuration
✅ **Docker Support** - Run in container or locally
✅ **Type Hints** - Fully typed Python codebase

## Tech Stack

- Python 3.11+
- FastAPI for web interface
- Docker for deployment
- Pytest for testing
- Pydantic for data validation

## Installation

```bash
pip install sample-project
```

## Usage

```bash
sample-process --input data/ --output report.json
```
