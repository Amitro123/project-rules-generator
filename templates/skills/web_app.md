### api-endpoint-analyzer
Analyze FastAPI/Flask endpoints for issues.

**When to use:**
- Before deployment
- Performance optimization
- Security audit

**Checks:**
- Missing authentication
- N+1 queries
- Large response sizes
- Missing rate limits

### frontend-backend-sync
Ensure API contract matches frontend expectations.

**When to use:**
- After API changes
- Before frontend deploys

**Output:** Compatibility report + migration guide
