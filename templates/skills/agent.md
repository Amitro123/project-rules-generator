### agent-workflow-analyzer
Analyze agent orchestration patterns and bottlenecks.

**When to use:**
- Complex multi-agent workflows
- Performance bottlenecks
- Debugging agent loops

**Tools:** read, exec, trace

### prompt-optimizer
Improve agent prompts and system messages for better output.

**When to use:**
- Poor agent output quality
- High API costs
- Inconsistent behavior

**Steps:**
1. Analyze current prompts
2. Test variations
3. Measure quality metrics
4. Implement best performer

**Usage:**
```bash
optimize-prompt --task "video-scene-description" --model gemini-pro
```

### llm-api-cost-analyzer
Track and optimize LLM API usage and costs.

**When to use:**
- Monthly costs increasing
- Need usage breakdown by feature
- Optimizing batch processing

**Output:** Cost report + optimization suggestions

### semantic-search-tuner
Improve search relevance and ranking.

**When to use:**
- Users report irrelevant results
- Need to tune embedding models
- Adjusting search parameters

**Tools:** vector DB, embeddings, eval metrics
